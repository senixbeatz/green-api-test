# GREEN-API Test Task

Тестовое задание на должность DevOps-разработчик.

Выполнил: **Павлов Арсений**

## Описание

HTML-страница для работы с методами [GREEN-API](https://green-api.com). Позволяет подключиться к WhatsApp-инстансу и вызывать API-методы прямо из браузера.

## Реализованные методы

- **getSettings** — получить настройки инстанса
- **getStateInstance** — получить состояние инстанса
- **sendMessage** — отправить текстовое сообщение
- **sendFileByUrl** — отправить файл по ссылке

## Как использоваться

1. Зарегистрироваться на [green-api.com](https://green-api.com) и создать инстанс
2. Отсканировать QR-код в личном кабинете, чтобы привязать номер WhatsApp
3. Открыть страницу, ввести `idInstance` и `ApiTokenInstance`
4. Нажать нужную кнопку — ответ отобразится в поле справа

## Технологии

Чистый HTML + CSS + JavaScript, без фреймворков и зависимостей.
